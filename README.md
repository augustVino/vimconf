gvimrc
======

the configuration file for vim on ubuntu 12.04
----------------------------------------------

## Quick start

1. 安装所需要的ctags和grep-ack

  ```
  $ sudo apt-get install ctags grep-ack
  ```

2. 安装vundle

  ```
  $ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
  ```

3. 在gvim中安装所有插件

  ```
  :BundleInstall
  ```
*Thank you!*
